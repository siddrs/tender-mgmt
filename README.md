# tender-mgmt
Databases Project - Course Work


### Tech Used: 
1. Streamlit (frontend + backend)
2. SQLite (DBMS)

### Features:
##### Admin:
1. Manage Tenders:
   - view all tenders -- filter by location, status
   - add new tender
   - delete open tender
   - edit open tender
2. Manage Vendors
   - view all vendors
   - add new vendor
   - delete a vendor
3. Manage Bids
   - view & evaluate bids
   - award winner
   - view logs
